# Akashdeep — Personal Research Portfolio

This repository hosts my personal GitHub Pages website:

https://a-akashdeep.github.io/

The website contains selected research projects, publications, technical experience, and links to supporting material. It is intended as a compact portfolio to complement my CV, especially for roles related to experimental physics, spintronics, materials science, data analysis, instrumentation, and R&D.

## Website sections

- Selected research and technical projects
- Publications and preprints
- Conference contributions
- CV and contact information

## Technical notes

The site is built with Jekyll and hosted using GitHub Pages. The design was adapted from the Academic Pages template, which itself is based on the Minimal Mistakes Jekyll theme. I have customized the content, structure, project pages, and presentation for my own academic and technical portfolio.
